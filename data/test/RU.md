## Выделение

_Этот текст будет курсивом_  
_Это тоже будет курсивом_

**Этот текст будет жирным**  
**Это тоже будет жирным**

_Можно **комбинировать** их_

## Списки

### Маркированный

- Элемент 1
- Элемент 2
- Элемент 2a
- Элемент 2b
  - Элемент 3a
  - Элемент 3b

### Нумерованный

1. Элемент 1
2. Элемент 2
3. Элемент 3
   1. Элемент 3a
   2. Элемент 3b

## Цитаты

> Markdown — это облегчённый язык разметки с синтаксисом форматирования обычного текста, созданный в 2004 году Джоном Грубером и Аароном Шварцем.
>
> > Markdown часто используется для форматирования readme-файлов, написания сообщений на форумах и создания форматированного текста в обычном текстовом редакторе.
